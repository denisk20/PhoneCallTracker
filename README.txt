ВНИМАНИЕ! 
Вам понадобится JAVA для запуска этой программы. JAVA может быть загружена здесь:
http://www.java.com/en/download/manual.jsp

Это программа предназначена для отслеживания и поиска телефонных звонков. Для запуска программы нужно запустить файл
run.bat из корневой папки программы. Рекомендуется время от временя сохранять содержимое папки db, в которой 
хранятся данные о звонках.
